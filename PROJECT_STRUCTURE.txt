================================================================================
  スポーツ観戦カレンダー - プロジェクト構造
================================================================================

│   │

│   ├── 📁 styles/
│   │   └── index.css                 ← グローバルスタイル
│   │

│   └── main.tsx                      ← エントリーポイント
│

│   └── 📁 data/

================================================================================
  主要なスクリプト
================================================================================

npm run dev                ← 開発サーバー起動（http://localhost:3000）
npm run build              ← 本番ビルド（tsc + Vite バンドル → dist/）
npm run type-check         ← TypeScript型チェック
npm run fetch-schedule     ← API から試合データ取得 → public/data/events.json

================================================================================
  データフロー
================================================================================

football-data.org API
      ↓
scripts/fetch-schedule.ts
      ↓
public/data/events.json
      ↓
src/hooks/useEvents.ts (フェッチ・フィルタリング)
      ↓
src/components/SportsCalendar.tsx (表示)

================================================================================
  主要な型
================================================================================

Event {
  id: string | number
  lg: "wc2026" | "jleague" | "intl" | "f1" | "olympic"
  date: string          // YYYY-MM-DD
  day: string           // 日・月・火・水・木・金・土
  time: string          // HH:MM
  title: string
  cat: string
  casts: string[]       // 放送局キー
  fav?: boolean
}

League {
  label: string         // 表示名
  color: string         // 16進カラーコード
}

Broadcaster {
  label: string         // 表示名
  kind: "配信" | "地上波" | "BS" | "CS"
  chip: string          // 16進カラーコード
  icon: LucideIcon
}
